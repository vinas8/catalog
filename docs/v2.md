**Decay Logic** (runs every hour):
```javascript
function applyHourlyDecay(snake) {
  const decayRates = getDecayRates(snake.species, snake.life_stage);
  
  snake.stats.hunger -= decayRates.hunger;
  snake.stats.water -= decayRates.water;
  
  // Temperature/humidity require equipment
  if (!hasWorkingHeater(snake.enclosure)) {
    snake.stats.temperature -= 2;
  }
  if (!hasWorkingMister(snake.enclosure)) {
    snake.stats.humidity -= 3;
  }
  
  // Stress naturally decreases
  snake.stats.stress = Math.max(0, snake.stats.stress - 1);
  
  // Health consequences
  if (snake.stats.hunger < 20) {
    snake.stats.health -= 5;
  }
  if (snake.stats.temperature < 75 || snake.stats.temperature > 95) {
    snake.stats.health -= 3;
  }
  if (snake.stats.humidity < snake.species.min_humidity) {
    snake.shed_cycle.stuck_shed_risk += 10;
  }
  
  // Death condition
  if (snake.stats.health <= 0) {
    snake.status = 'deceased';
    triggerDeathEvent(snake);
  }
}
```

```javascript
function breedSnakes(parent1, parent2) {
  const offspring = [];
  const clutchSize = randomInt(4, 12);
  
  for (let i = 0; i < clutchSize; i++) {
    const baby = {

function inheritTraits(p1_genes, p2_genes) {
  // Punnett square logic
  const result = {};
  for (const trait of allTraits) {
    const p1_allele = randomChoice([p1_genes[trait].allele1, p1_genes[trait].allele2]);
    const p2_allele = randomChoice([p2_genes[trait].allele1, p2_genes[trait].allele2]);
    result[trait] = { allele1: p1_allele, allele2: p2_allele };

  return result;
```

## âš ï¸ RISKS & MITIGATION

### Business Risks
1. **Low Conversion Rate**
   - Risk: Game users don't buy snakes
   - Mitigation: Free virtual snakes, but incentivize real purchase (exclusive morphs)

2. **CAC > CLV**
   - Risk: Expensive to acquire users
   - Mitigation: Focus on organic (SEO, referrals), viral loops

3. **Partner Farms Disintermediate**
   - Risk: Farms go direct after getting customers
   - Mitigation: Platform exclusives, analytics, community value

### Technical Risks
1. **Scale/Performance**
   - Risk: Slow app with thousands of users
   - Mitigation: Cloudflare edge, caching, CDN

2. **Data Loss**
   - Risk: Users lose their collection (save corruption)
   - Mitigation: Cloud backups, redundancy, testing

3. **Security**
   - Risk: Payment fraud, account hacking
   - Mitigation: Stripe (PCI compliant), 2FA, rate limiting

### Legal/Regulatory Risks
1. **Animal Welfare**
   - Risk: Accused of encouraging impulse purchases
   - Mitigation: Education-first, waiting periods, care verification

2. **COPPA (Children <13)**
   - Risk: Fines for collecting data from kids
   - Mitigation: Age gate, parental consent flow

3. **State Regulations**
   - Risk: Some states restrict snake sales/shipping
   - Mitigation: Geo-blocking, compliance checks
