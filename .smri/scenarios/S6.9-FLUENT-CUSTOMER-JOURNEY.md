```javascript
// Unified snake management
class SnakeManager {
  getAll() {
    return {
      real: this.getRealSnakes(),
      virtual: this.getVirtualSnakes()
    };
  }
  
  getRealSnakes() {
    // From USER_PRODUCTS KV
    return userProducts.filter(p => p.type === 'real');
  }
  
  getVirtualSnakes() {
    // From localStorage
    return JSON.parse(localStorage.getItem('virtual_snakes') || '[]');
  }
  
  // Same care interface for both!
  feed(snakeId) {
    const snake = this.getSnake(snakeId);
    snake.hunger += 30;
    snake.lastFed = Date.now();
    this.save(snake);
    
    // Show same animation regardless of type
    showFeedAnimation(snake);
  }
}
```

**Key insight:** User shouldn't think "real vs virtual" - just "my snakes"
